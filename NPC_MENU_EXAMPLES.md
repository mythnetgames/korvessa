## Mass Production Workflow

### Step 1: Design Once

```
/designnpc
→ Gang Boss
  Stats: body:5, ref:5, dex:5, tech:4, smrt:5, will:5, edge:4, emp:6
  Skills: brawling:50, blades:45, ranged:40, grapple:45, dodge:40, etc.
  ID: npc_42
```

### Step 2: Spawn Multiple Copies

```
/spawnnpc npc_42
→ Spawned: Gang Boss (ID: 234)

/spawnnpc npc_42
→ Spawned: Gang Boss (ID: 235)

/spawnnpc npc_42
→ Spawned: Gang Boss (ID: 236)
```

### Result: 3 Identical NPCs

All three have:
- Same stats (5, 5, 5, 4, 5, 5, 4, 6)
- Same skills (50, 45, 0, 40, 45, 40, 30, 60, 55, 45)
- Same combat effectiveness
- Different in-game IDs (different objects)
- Can be individually renamed/customized if needed

---

**This system makes it easy to create balanced, mass-produced NPCs for quests and encounters!**
